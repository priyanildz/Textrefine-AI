# TextRefine AI

AI-based communication skill improver that helps users enhance their writing with:
- Paraphrasing
- Grammar Correction
- Spelling Correction
- Sentiment Analysis

---

## Features

- AI-powered text analysis  
- Grammar & paraphrasing suggestions  
- Sentiment detection  
- User authentication (Login/Register)  
- Clean and modern UI
